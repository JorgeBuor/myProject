Este proyecto consiste en el desarrollo de un sistema de software que permite calcular la forma de pago de las órdenes de compra de una empresa contratante que adquiere piezas de un fabricante. Dependiendo del monto total de la compra, el sistema determina cómo se distribuye el pago entre inversión propia, préstamo bancario y crédito al fabricante, considerando un interés del 20% sobre el monto financiado.
El sistema ha sido desarrollado en Python y gestionado con Git para el control de versiones, permitiendo un desarrollo organizado mediante el uso de ramas para la implementación de nuevas funcionalidades. La estructura del proyecto incluye un archivo principal main.py, donde se encuentra la lógica de cálculo, un readme.txt con la documentación del proyecto y una carpeta docs/ para la configuración y referencias técnicas.
El desarrollo sigue buenas prácticas de programación y control de versiones, garantizando modularidad y facilidad de mantenimiento. Además, el código se ha subido a GitHub para su almacenamiento y colaboración remota. Este sistema facilita la gestión financiera de compras en la empresa, optimizando el cálculo y distribución de pagos de manera eficiente.
